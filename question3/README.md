# Question 3 – React Blog UI

This mini project is a simple blog user interface developed using React.

## Objective
The objective of this project is to create a blog layout that displays posts from a JSON file and allows users to search and filter posts.

## Features
- Displays blog posts in card layout
- Reads post data from a JSON file
- Search functionality to find posts by title
- Filter posts by category
- Reusable React components

## Technologies Used
- React
- JavaScript
- HTML
- CSS

## How to Run the Project

1. Create a React project using create-react-app
2. Copy these files into the src folder
3. Install dependencies

npm install

4. Run the project

npm start

The application will run on:
http://localhost:3000
